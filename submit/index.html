We encourage submissions from scientists at all career stages for both posters and talks. To submit an abstract:

  1. Download the abstract template.
  2. Enter your abstract text and author information, following the format laid out in the template.
  3. Delete the instructions (in red) from the template.
  4. Save the template as a PDF file with file name "YourLastName_MWSE50.pdf".
  5. Email the abstract PDF to TurkeyRunPhotosynthesis@gmail.com.
